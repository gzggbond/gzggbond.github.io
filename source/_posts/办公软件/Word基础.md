# 技巧

1. 光标选中文字区域开头，按住`Shift`，鼠标点击结尾可选取所有区域

   按住`Ctrl`加鼠标可以选取多个部分

   按住`Alt`加鼠标可以矩形选中区域

2. 格式刷选择时不要选中段落标记即可只复制文字格式

   段落任意处选中格式刷，在另一个段落的非文字处点击即可复制段落格式

3. `Word`表格的相关调整都是在“布局”处【点击表格才会出现的“布局”】，表格删除选中数据使用`delete`键

   表格布局的属性处可以处理换行，表格内容显示不全等问题

## 快捷键

+ `Ctrl + home/end`：回到文档开头/结尾
+ `Shift + 回车`：换行符【回车是分段符，同一个段一般格式会一样，看自己需要】
+ `Ctrl + L/E/R`：左对齐`/`居中`/`右对齐
+ `F4`：重复上一步操作【笔记本的话需要加上`Fn`键】
+ `方向键x + 回车`：在当前光标处的`x`方向插入回车符
+ `Alt + Shift + 方向键x`：将当前光标所在单元格进行方向`x`移动
