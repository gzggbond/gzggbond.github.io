### 5.5.1 日期

> `Excel`中的日期标准格式是`/`分割

### 5.5.2 百分比

> 可以在`“开始”`的`“数字”`部分将整列设置为百分比格式，接着直接输入数字再回车即可得到百分比格式
>
> 小数点位数也可以通过`“数字”`部分的小标进行调整

### 5.5.3 分数

> 用/表示分数会被识别为日期，因此在输入分数前需要先输入`0空格`
>
> 小数转为分数只需要选取一整列，接着在`“数字”`部分标记为分数即可

```c
// 这样输入到单元格中会被识别为1/2
0 1/2
```

### 5.5.4 文本

> 超过`15`位的数字在`Excel`中无法正常显示，因此需要调整为文本格式，一种方法是在输入前设置输入框为文本格式，另一种则是在数字最前边输入`'`转换为文本格式

```c
// 转换为文本格式，使其能在单元格正常显示
'222222222222222222222
```

### 5.5.5 自定义

#### 数值占位符0,#,?

<img src="https://z-cloud-pic-1313046262.cos.ap-guangzhou.myqcloud.com/img/202304291945063.png" alt="image-20230429194531968" style="zoom: 67%;" />

#### 文本占位符@

> 作用于文本的占位符，可以用英文引号""中加入文本，在文本前后增加数值对文本进行自定义格式

#### 数字格式4区域

`0;-0;0;@`：利用数字格式`4`个区域的定义（正数;负数;零值;文本），可以对各区域内容独立定义

#### 颜色

颜色代码必须在代码部分中的第一个项目颜色的两边必须加上英文方括号`[]`

可供选择的颜色代码有：`[黑色][白色][红色][绿色][蓝色][黄色][洋红色]`

#### 比较运算符<=>

在实际代码应用中，只能在前两个区域中使用`">"、"<"、"="`等条件代码，第三个区域以除此外作为条件值

+ `[条件1];[条件2];[其他]`
+ `[条件];[其他]`

#### 日期

`y-m-d`：通过改变代码的数量和顺序改变日期的显示方式

## 7.1 Count函数家族

+ `Count`：**返回参数列表中的==数字==的单元格的个数**。利用`COUNT`函数可以计算单元格区域或数字数组中数字字段的输入项个数

+ `CountA`：**返回参数列表中==非空==值的单元格个数**。利用函数`COUNTA`可以计算单元格区域或数组中包含数据的单元格个数

+ `CountBlank`：**返回参数列表中==空==值的单元格个数**。利用函数`COUNTBLANK`可以计算单元格区域或数组中包含空格的单元格个数

+ `CountIf`：有单一条件的计数【如果要实现**或**，则需要多个`CountIf`组成表达式】

  通配符`*，?，~`：分别代表任意多个字符，一个字符和转义字符【过滤计数文本信息时会用到】

+ `CountIfs`：有多种条件的计数

## 7.2 数学函数

+ `Sum(num1,num2)`：`Number`**求和的数值**，可以将单个值、单元格引用或是区域相加，或者将三者的组合相加

  <img src="https://z-cloud-pic-1313046262.cos.ap-guangzhou.myqcloud.com/img/202305011509016.png" alt="image-20230501150919915" style="zoom:67%;" />

  `SumIfs`类似，只是可以填写更多条件

+ `Average(num1,num2)`：要计算平均值的数字、单元格引用或单元格区域

  > `average`也有`if`和`ifs`

+ `Rank`函数可以实现排序

## 功能

```c
// 快速填充
ctrl + E

Ctrl + Q
// 显示当前日期

Alt + =
```
