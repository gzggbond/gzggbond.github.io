```java
// 跳转到下一行
Ctrl + Enter		
// 整理代码格式
shift + alt + f
// 回到上/下一步光标位置
alt + 左/右箭头	
// 选中当前行
shift + alt 左/右箭头
// 复制当前行到上/下面
shift + alt 上/下箭头
// 全局查找并替换单词
ctrl + h
// 快速定位到某行
Ctrl + g
// 全局放大VScode界面
Ctrl + '+/-'
// 开启多个光标
Ctrl + alt + 上/下箭头
// 左右快速移动光标
Ctrl + 左/右箭头
// 左右快速移动光标并选中
Ctrl + shift + 左/右箭头
// 快速定位到行首/末
Home/end
```
